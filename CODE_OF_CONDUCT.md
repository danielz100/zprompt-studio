# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**Standards**: be respectful, assume good faith, focus on technical merit, accept constructive feedback.

**Not tolerated**: harassment, personal attacks, discriminatory language, sustained disruption.

**Enforcement**: report incidents privately to **danielzumarraga999@gmail.com**. Reports stay confidential. Maintainers may warn, ban, or remove contributions per severity.

**Scope**: applies in all project spaces — repo, issues, PRs, Discussions — and when representing the project publicly.
